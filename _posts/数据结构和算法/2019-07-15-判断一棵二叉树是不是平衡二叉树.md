## 题目描述
输入一棵二叉树，判断该二叉树是否是平衡二叉树。

```java
public static class TreeNode {
         int val = 0;
         TreeNode left = null;
         TreeNode right = null;
         public TreeNode(int val) {
             this.val = val;
         }
     }
```

使用递归，从下往上遍历，如果子树是平衡二叉树，就返回子树的高度；如果子树不是平衡二叉树，返回-1停止遍历。最后判断如果结果是-1，就说明不是平衡二叉树。

```java
public class test {
     public static class TreeNode {
         int val = 0;
         TreeNode left = null;
         TreeNode right = null;
         public TreeNode(int val) {
             this.val = val;
         }
     }
     public static boolean IsBalanced_Solution(TreeNode root) {
           //true or false
        return Traversal(root) != -1;
    }
     public static int Traversal(TreeNode root){
           //如果节点为空，子树高度为0
           if(root == null){
                return 0;
           }
           //接收当前节点左子树的返回值。如果得到-1，说明已经不是平衡二叉树，继续返回-1
           int left = Traversal(root.left);
           if(left == -1){
                return -1;
           }
           //同上
           int right = Traversal(root.right);
           if(right == -1){
                return -1;
           }
           //如果一个子树根节点的左右子树的高度差大于1，说明该子树不是平衡二叉树，返回-1。
           if(left - right >1 || right - left > 1){
                return -1;
           }
           //如果该子树是平衡二叉树，返回该子树的高度
           else{
                return (left > right ? left : right) + 1;
           }
     }
}
```
